## 第二步：形态选择——笔记本 vs 台式机

- **笔记本 (Laptop)**

  - ✅ **优点**：便携，自带屏幕键盘电池，甚至可以床上用。
  - ❌ **缺点**：同价位性能不如台式机，散热受限，升级空间小，维修成本高。
  - 👉 **适合**：大学生、经常出差、租房党、桌面空间有限者。

- **台式机 (Desktop)**
  - ✅ **优点**：同价位性能最强，散热好，噪音低，升级方便，显示器可以很大。
  - ❌ **缺点**：带不走，需要外设，断电即关机。
  - 👉 **适合**：家庭固定使用、纯游戏玩家、专业工作室、追求极致性价比。

## 第三步：读懂核心参数（拒绝被忽悠）

### 1. CPU (中央处理器) - 电脑的大脑

- **阵营**：Intel (英特尔) vs AMD (锐龙)
- **等级划分**：
  - **入门**：i3 / R3 （仅限极低预算或纯办公）
  - **主流**：i5 / R5 / Core Ultra 5 （**推荐起点**，性价比最高）
  - **高端**：i7 / R7 / Core Ultra 7 （高性能需求）
  - **顶级**：i9 / R9 / Core Ultra 9 （不差钱，生产力）
- **⚠️ 注意**：**不要只看 i5/i7！** 一定要看**代数**。例如 `i5-13500H` 强于老款 `i7-7700HQ` 无数倍。认准当年最新或次新一代架构。

### 2. GPU (显卡) - 游戏与图形的核心

- **轻薄本**：通常使用**核显** (Intel Arc Graphics / AMD Radeon 780M 等)，能玩轻度网游，省电。
- **游戏/全能本**：必须有**独显** (NVIDIA RTX 系列)。
  - **入门**：RTX 4050 (性价比高，畅玩 1080P)
  - **中阶**：RTX 4060 (甜品级，**最推荐**，通吃 2K 主流游戏)
  - **高端**：RTX 4070 / 4080 / 4090 (发烧友，4K 游戏，专业渲染)
- **⚠️ 注意**：笔记本显卡要看**功耗释放 (TGP)**。同样是 4060，140W 满血版和 45W 残血版性能差距巨大！

### 3. RAM (内存) - 决定多任务能力

- **标准**：**16GB** 是目前的底线。
- **推荐**：**32GB** （强烈推荐，未来两三年不卡顿，Chrome 开几十个网页无压力）。
- **避坑**：现在的轻薄本很多是**板载内存**（焊死的，不能升级），买的时候建议一步到位选大内存版。

### 4. Storage (硬盘) - 存东西的地方

- **类型**：必须是 **NVMe SSD (固态硬盘)**。机械硬盘(HDD)只能用来做仓库盘。
- **容量**：**512GB** 勉强够用（装几个大游戏就满了），**1TB** 是目前标配，建议 2TB 更从容。

### 5. Screen (屏幕) - 眼睛的体验

- **分辨率**：拒绝 1080P 低色域屏！推荐 **2.5K (2560x1600)** 或以上。
- **色域**：必须 **100% sRGB** 或 100% DCI-P3。低色域屏画面发白，千万别买。
- **刷新率**：游戏本需 **144Hz** / **165Hz** / **240Hz**；办公本 60Hz-90Hz 即可，但高刷确实丝滑。

## 第四步：避坑指南 & 购买注意事项（⭐ 重点）

### 🚨 1. 实体店“转型”骗局 (经典套路)

- **场景**：你去实体店点名要买型号 A。
- **话术**：“哎呀，A 这款刚好没货/这款返修率特别高/散热很差/玩游戏卡顿。你要不看看 B 款？配置更高，价格还便宜。”
- **结果**：B 款通常是几年前的库存垃圾，或者低配高价的智商税机型。
- **对策**：**网购！网购！网购！** 实在要去实体店，咬定型号不松口，没货就走人，不要听销售忽悠任何“同价位更好”的推荐。

### 🚨 2. 电商“特供版”与文字游戏

- **场景**：某多多或直播间价格异常低。
- **陷阱**：可能是“特供版”，屏幕缩水（低色域）、网卡缩水、硬盘是杂牌、或者是翻新机/退货机。
- **对策**：认准 **京东自营** 或 **天猫品牌官方旗舰店**。

### 🚨 3. 二手水深勿入

- 除非你是硬件老手，否则**不要碰闲鱼二手笔记本**。暗病、进水修过、矿本（用来挖矿的显卡）、拼装机防不胜防。

### 🚨 4. 伪需求陷阱

- **触摸屏**：在 Windows 笔记本上，99% 的人用了两次就不再用了，只会增加重量和镜面反光。
- **4K 屏幕 (在小尺寸笔记本上)**：13-15 寸屏幕上 4K 和 2.5K 肉眼差别不大，但 4K 会严重拖累续航和显卡性能。

### 🚨 5. 软件与激活

- **Office**：现在的正版 Windows 笔记本通常预装 **Office 家庭和学生版** (Word, Excel, PPT)，激活后**永久免费**。不要花钱去买盗版或者是销售推销的“专业版”。
- **杀毒软件**：Windows 自带的 **Defender** 已经足够好用。**不需要**安装 360、电脑管家、金山毒霸等，它们往往是电脑卡顿的元凶。

## 附录：收货验机简易流程

1.  **检查外箱**：是否有严重破损、受潮。查看封条是否完好。
2.  **核对序列号**：外箱上的 S/N 码应与机身底部的 S/N 码一致。
3.  **外观检查**：
    - 划痕、磕碰、螺丝是否有拧过的痕迹。
    - 转轴是否异响，键盘按键是否回弹正常。
4.  **开机 (关键一步！)**：
    - **此时千万不要连接 WiFi！** **此时千万不要连接 WiFi！**
    - 大部分品牌笔记本，一旦联网激活 Windows，就不支持**七天无理由退货**了。
    - 先跳过联网步骤进系统（部分 Win11 版本需要特殊命令跳过，可搜教程）。
5.  **屏幕测试**：使用纯色图片检查是否有**坏点**或严重**漏光**。
6.  **配置核对**：打开任务管理器，确认 CPU、内存、显卡型号是否与购买一致。
7.  **确认无误后**：连接 WiFi，激活系统，开始享受你的新电脑！
